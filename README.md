# 💰 Python Expense Tracker

A simple Python Expense Tracker that allows users to enter multiple expenses and calculates the total amount spent. This project was created as part of the DecodeLabs Python Internship - Project 2.

## 📌 Features

- Add multiple expenses
- Calculate total expenses automatically
- Display expense summary
- Calculate average expense
- Handle invalid input
- User-friendly console interface

## 🛠️ Technologies Used

- Python 3
- Google Colab / VS Code

## ▶️ How to Run

1. Open the project in Google Colab or any Python IDE.
2. Run the Python file.
3. Enter expense amounts.
4. Type `done` to finish.
5. View the expense summary.
